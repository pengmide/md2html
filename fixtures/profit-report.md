## 1. 管理层摘要 📊

- 本次覆盖 16 个管理区域。**门店毛利额同比 9 升 7 降、环比 12 升 4 降**：环比层面多数区域较 4 月修复，但同比层面盈利分化明显，承压区域集中在 **运营中心直管、西安区、香港区、南京区**，以及补贴抬升但门店毛利反而下滑的 **成都区、重庆区**。
- 盈利质量整体呈「**少数区域稳健增长、多数区域利润承压、部分区域补贴与折扣依赖**」格局。稳健增长以 **澳门区（门店毛利额同比 +9.93%、补贴额 0）**、**粤东区（同比 +7.09%、补贴额同比 -19.54%）** 为代表；承压最重的是 **运营中心直管（门店毛利额同比 -25.25%、19 点前销售额同比 -14.87%）**，呈需求与利润同步收缩。
- 盈利差异主要体现在三类证据：① **补贴依赖**——成都区、重庆区补贴额同比 +156.44% / +124.73%，但门店毛利额同比 -4.73% / -2.35%，补贴后毛利额却同比 +4.35% / +4.8%，补贴在「托住」补贴后口径、却未托住门店口径；② **折扣侵蚀**——华东区时段折扣率 16.73%、促销折扣率 15.05% 全域最高，门店毛利额同比 -1.26%；③ **供给未转化**——西安区进货重量同比 +21.01%，但 19 点前销售额同比仅 +0.32%、门店毛利额同比 -13.15%。
- 当前最优先动作：对 **运营中心直管** 启动客流与毛利双重止损（19 点前客数同比 -9.54%、门店毛利额同比 -25.25%），对 **西安区 / 成都区 / 重庆区** 复盘补贴效率与折扣结构，避免「补贴涨、门店利润跌」继续扩大。

## 3. 同比环比归因总览 🔍

围绕核心盈利指标的同比/环比方向，跨层归因如下（仅使用 CLI 已返回的增长率字段）：

- **利润结果**：门店毛利额与补贴后毛利额在多数区域同向，但 **重庆区、成都区出现「补贴后毛利额同比上升、门店毛利额同比下降」的反向**——补贴后口径改善完全由补贴额驱动，门店端实际盈利在恶化。西安区两者同比双降（-13.15% / -9.64%），补贴未能止住下滑。
- **毛利空间**：门店预期销售毛利率与定价毛利率仅返回当期水平值、未返回同比/环比字段，因此**不判断毛利率同比方向**；仅以当期水平辅助解读。西安区（33.47% / 30.95%）、武汉区（37.04% / 33.63%）、运营中心直管（37.82% / 34.67%）毛利水平偏低，与其利润下滑并存。
- **补贴与折扣**：补贴额同比上升的区域中，西安区（+3566.44%，小基数）、成都区（+156.44%）、重庆区（+124.73%）、天津区（+168.67%）、华东区（+13.23%）、运营中心直管（+53.44%）均伴随门店毛利额同比下滑，提示补贴效率存疑；折扣率（时段/促销）未返回同比/环比字段，仅以当期水平结合利润同比判断（见第 5 节）。
- **供给与订购**：进货重量同比上升但门店毛利额同比下降的区域有西安区（+21.01% → 利润 -13.15%）、重庆区（+20.43% → -2.35%）、成都区（+18.01% → -4.73%）、华东区（+7.76% → -1.26%），供给投入未有效转化为门店利润。
- **客单与 19 点前时段**：19 点前销售额同比下降且门店毛利额同比下降的区域为运营中心直管（-14.87% → -25.25%）、香港区（-8.5% → -7.2%）、南京区（-9.83% → -7.05%）、华东区（-0.33% → -1.26%），主时段质量与利润同步承压。

## 口径说明 📌

- **时间范围**：当期 `--start-date 2026-05-01 --end-date 2026-05-31`（当前日期 2026-06-26，「上个月」按上一个自然月首日至末日）；同比 = 2025-05 同口径，环比 = 2026-04 同口径，由 CLI 原生 `--yoy --mom` 计算并合并到当期行的 `*同比增长率` / `*环比增长率` 字段。

- **统计口径**：店日均 `storeCollectType=1`；金额、重量均为店日均口径数值，比率指标带 `%` 者为 JSON 原值。
- **区域过滤**：未追加区域过滤，使用 CLI 默认可见范围。
- **品类过滤**：未追加品类过滤，未做品类列维度拆解。
- **指标列表**：receiveWeight、bf19CustNum、onlineCustNum、hourDiscountRate、promotionDiscountRate、allowanceAmt、allowanceProfitAmt、preProfitRate、prePriceProfitRate、bf19SaleWeight、profitAmt、storeOrderAmt、storeOrderWeight、orderArticleNum、bf19SaleAmt、receiveAmt、perCustAmt、bf19PerCustAmt（共 18 个指标）。

- **同比/环比来源**：CLI 原生 `--yoy --mom` 合并到当前行的 `*同比增长率` / `*环比增长率` 字段，报告侧未重新计算，未补造基准期原值。
